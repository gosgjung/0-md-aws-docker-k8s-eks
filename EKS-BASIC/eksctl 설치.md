### Windows

관리자 권한으로 터미널/파워셸을 실행 후 아래의 명령들을 실행



install

```bash
$ choco install -y eksctl 
```

upgrade

```bash
$ choco upgrade -y eksctl 
```

설치 확인

```bash
$ eksctl version
```
